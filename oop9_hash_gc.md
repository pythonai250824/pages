## תרגיל – מחלקת נקודה (Point)

צור מחלקה בשם `Point` שמייצגת נקודה דו־ממדית עם קואורדינטות `x` ו־`y`.

#### דרישות:

1. בקונסטרקטור (`__init__`) קבל את הערכים `x` ו־`y`, ושמור אותם כמאפיינים פרטיים.
2. כתוב **getter** ו־**setter** לשני המאפיינים.
3. מימוש מלא של:

   * `__repr__`
   * `__str__`
   * `__eq__`
   * `__hash__`
4. צור שתי נקודות עם אותם ערכים (למשל, `Point(3, 4)` ו־`Point(3, 4)`).
5. שמור ערך כלשהו במילון כשהמפתח הוא האובייקט הראשון.
6. הוסף ערך נוסף כשהמפתח הוא האובייקט השני.
7. הדפס את המילון.

#### שאלות חשיבה:

* כמה ערכים נשמרים בפועל במילון? למה?
* מהו hash collision? האם זה יכול לקרות כאן?
* מהו reference count בזיכרון? איך זה קשור ל־GC (garbage collection)?
